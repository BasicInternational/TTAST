## 설치 
  1. git clone https://github.com/BasicInternational/TTAST.git
  2. cd tools/
  3. npm install --global npm@latest
  4. npm install 

## build
  1. cd tools/
  2. npm run build

## Start the localhost server
  1. cd tools/
  2. npm run localhost
